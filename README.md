# HomeWork5

Все поля структур сделаны приватными (с маленькой буквы)

Для доступа и изменения используются методы (геттеры и сеттеры)

Например, GetName(), SetName(), GetSize(), SetSize()


Структура пакетов:

internal/model/player - для игроков

internal/model/game - для игр и ходов

internal/model/board - для игрового поля


Сущности:

Player - игрок с именем

Game - игра с размером, ходами, состоянием

Move - ход с координатами и фигурой

Piece - фигура с типом, цветом и символом

Board - доска с клетками

Cell - клетка с позицией и фигурой

Фабрика: SetupInitialBoard создает начальную расстановку фигур

Основная функция main использует новые структуры вместо глобальных переменных
